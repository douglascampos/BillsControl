== README
A idéia desse projeto é criar um app para cuidar das contas pessoais, já estou cansado de procurar aplicativos para fazer o que eu quero que seja feito.
Decidi criar o meu.

## Para utilizar
* Baixar o código $ git clone

* Criar o banco $ rake db:create

* Criar as tabelas $ rake db:migrate

* Subir o projeto $ rails server
